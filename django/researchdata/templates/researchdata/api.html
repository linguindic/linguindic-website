API

To support open research, this project allows you to interact with the research data via its open API (Application Programming Interface). This can make accessing and working with the data much easier than through the standard web interface for users wishing to incorporate the data in their own software.

The following API endpoints are available in this project:

  • /api/linguisticnotions/
  • /api/linguisticnotions/<id>

For more information about each API please visit the specific endpoint's url in your web browser.

For any queries using our API please contact us at linguindic@orinst.ox.ac.uk
